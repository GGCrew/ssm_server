Install Debian (default Raspberry Pi installation)

Install required Server software
	Server commands:
		apt-get install
			build-essential
			libssl-dev
			libreadline-dev
			libmysqlclient-dev
			libfreeimage-dev
			libyaml-dev
			mysql-server
			
		Build node.js using these instructions: (https://github.com/joyent/node/wiki/Installing-Node.js-via-package-manager)
			sudo apt-get install python g++ make
			mkdir ~/nodejs && cd $_
			wget -N http://nodejs.org/dist/node-latest.tar.gz
			tar xzvf node-latest.tar.gz && cd `ls -rd node-v*`
			./configure
			sudo make install			

			
		Install Ruby
			cd ~
			wget ftp://ftp.ruby-lang.org/pub/ruby/ruby-1.9-stable.tar.gz
			tar -zxvf ruby-1.9-stable.tar.gz
			cd ruby-1.8.7-p72
			./configure 
			make
			make install
			
		
		Install RubyGems
			cd ~
			wget http://rubyforge.org/frs/download.php/76073/rubygems-1.8.24.tgz
			tar -zxvf rubygems-1.8.24.tgz
			cd rubygems-1.8.24
			ruby ./setup.rb
		
		
		gem install
			mysql
			rails
			
			
Create a new application
	rails new SnapShowMagicServer --database=mysql
